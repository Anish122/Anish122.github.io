  • Things To DoOpen or Close

    when we go to the market we often make a list of essential items to get from the market so that we would not miss anything.
    This experiment does the same thing but in a different and technical way. Basically it contains a input box to take data from user and a submit button. As long as user add items in the list, the list will get updated automatically. If user want to delete any entry, he/she can do it by just clicking on the entry which is appearing in the list.
    After making the list, user can either save it to his/her computer or email it to any email account.

    technologies used: Javascript, HTML, CSS

  • GCM for AndroidOpen or Close

    Google Cloud Messaging for Android (GCM) is a service that helps developers send data from servers to their Android applications on Android devices. Using this service you can send data to your application whenever new data is available instead of making requests to server in timely fashion. Integrating GCM in your android application enhances user experience and saves lot of battery power.
    Technology Used: Php
    For more information and to download the source code, visit my github account.
